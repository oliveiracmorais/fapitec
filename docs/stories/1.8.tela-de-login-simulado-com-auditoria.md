# Story 1.8: Tela de Login Simulado com Auditoria em Memória

## Status

Deprecated — Substituída pela Story 1.9 (backend autenticação), Story 1.10 (ajustes de conformidade) e Story 1.11 (recuperação de senha). Manter como referência para requisitos de UI da tela de login.

## Descrição
Implementar tela de login simulada usando dados em memória para fins de demonstração à FAPITEC-SE. O sistema deve validar CPF/Passaporte e senha, registrar auditoria mínima e permitir acesso a diferentes perfis de usuário, tudo sem persistência permanente.

## Contexto
Esta story complementa a demonstração existente (Story 1.6) substituindo o fluxo de identidades demonstrativas por um login autêntico simulado. A abordagem em memória é adequada para fins de demonstração e validação institucional, eliminando complexidade de infraestrutura.

## Tarefas

- [ ] Criar array em memória para usuários fake
- [ ] Implementar validação de CPF e passaporte
- [ ] Criar tela de login com tema FAPITEC
- [ ] Implementar 3 usuários fake com perfis distintos
- [ ] Criar auditoria mínima em memória (array de eventos)
- [ ] Implementar mensagens de erro conforme especificações
- [ ] Adicionar fluxo de "esqueci minha senha" (placeholder)
- [ ] Adicionar link para cadastro (placeholder)

## Critérios de Aceitação

- [ ] Formulário de login com campos CPF/Passaporte e senha
- [ ] Validação de formato CPF (XXX.XXX.XXX-XX) ou passaporte
- [ ] Validação de senha: mínimo 8 caracteres, letras+números+símbolos
- [ ] Mensagem de erro exata: "CPF/Passaporte ou Senha inválidos. Tente novamente."
- [ ] 3 usuários funcionais: Administrador, Proponente, Bolsista
- [ ] Auditoria registra em memória: timestamp, usuário, IP simulado, resultado
- [ ] Tela usa imagens institucionais FAPITEC
- [ ] Login bem-sucedido redireciona para dashboard
- [ ] Login falho mantém na tela com mensagem de erro
- [ ] Botão "Esqueci minha senha" funcional (placeholder)
- [ ] Link "clique aqui para cadastro" funcional (placeholder)
- [ ] Dados são armazenados apenas em memória (sem persistência)
