# Story 1.14: Frontend Gestão de Editais

## Status

Done

## Story

**Como** usuário da plataforma FAPITEC-SE,
**quero** acessar a gestão de editais pelo navegador,
**para que** possa visualizar, criar e acompanhar editais de pesquisa e inovação.

## Acceptance Criteria

1. A página `/editais` deve listar todos os editais cadastrados em formato de tabela.
2. A listagem deve permitir filtrar por título e status.
3. Deve haver um botão "Criar Edital" que leva ao formulário de criação.
4. O formulário de criação deve conter campos para nome, descrição, data início, data fim e tipo de chamada.
5. Após criar, o usuário deve ser redirecionado à listagem.
6. Deve ser possível visualizar os detalhes de um edital clicando nele.
